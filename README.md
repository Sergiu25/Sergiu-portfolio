# Sergiu-Catalin Petris Portfolio

Personal portfolio website built to present my background, skills, work experience, projects and contact information.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Features

- Responsive portfolio layout
- Backend-focused profile and personal introduction
- Skills grouped by area
- Work experience section
- Personal projects with GitHub links
- Contact section with email form, GitHub, LinkedIn and CV download

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
